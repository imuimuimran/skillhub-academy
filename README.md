# SkillHub Academy

A modern online learning platform built using Next.js App Router and Firebase Authentication.

## Features

- Responsive Landing Page
- Course Catalog
- Search & Filtering
- Dynamic Course Details
- Firebase Authentication
- Google Login
- Protected Routes
- Add Courses
- Manage Courses
- Responsive Design

## Technologies

- Next.js 15
- React
- Firebase Authentication
- Tailwind CSS
- React Hot Toast

## Installation

```bash
git clone <repository-url>

cd project-name

npm install

npm run dev
```
